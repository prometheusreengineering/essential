## Features
- Unlocks ALL cosmetics and emotes.
- Dumps ALL cosmetics and emotes data into `.minecraft/prometheus/dumps/essential/`.
- Lightweight and non-invasive.
- Compatible with *almost* all versions of Essential Mod.
